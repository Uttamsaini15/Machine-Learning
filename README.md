# 🤖 Machine Learning Journey

*   **02_Supervised_Learning/** - Regression, Classification, SVMs, Decision Trees, etc.
*   **03_Unsupervised_Learning/** - Clustering (K-Means, Hierarchical) and Dimensionality Reduction (PCA).
*   **04_Deep_Learning_&_Neural_Networks/** - Architectures built using TensorFlow/PyTorch.
*   **05_Projects/** - End-to-end ML workflows, data preprocessing pipelines, and model deployments.
